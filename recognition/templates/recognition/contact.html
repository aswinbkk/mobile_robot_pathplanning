About Us

This project involves building an automatic gate opening system that uses facial recognition to authenticate users. It covers areas such as facial detection, alignment, and recognition, along with the development of a web application to cater to various use cases of the system, such as user login, the addition of a new face to the training database, etc. This project intends to serve as an efficient substitute for gate opening systems. It can be used in corporate offices, schools, and organisations where security is essential. This project aims to automate the existing gate opening system, where the gate is currently unlocked manually. Its new features are an effective upgrade and replacement for the existing gate opening system.
